### <a name="tocol"></a> _static_ toCol

```js
// static toCol(col: number): string

/**
 * Converts number to column letter in A1.
 */

// Example
A1.toCol(1); // 'A'
A1.toCol('A1'); // Error
```

### <a name="torow"></a> _static_ toRow

```js
// static toRow(row: number): string

/**
 * Converts number to row string in A1.
 */

// Example
A1.toRow(1); // '1'
A1.toRow('A1'); // Error
```
